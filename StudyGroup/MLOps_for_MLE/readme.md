# MLOps

### 1. Introduction
  - ML Dev(머신러닝 모델 개발) + Ops(머신러닝 운영) = MLOps
  - 머신러닝 모델 개발과 머신러닝 운영 workflow의 간극을 줄이기 위한 분야
  - 머신러닝을 운영하는 데 기반이 되는 소프트웨어, 인프라, 배포, 개발 방법론 등의 전반적인 것을 아우르는 플랫폼
  - 머신러닝 라이프 사이클을 관리할 수 있는 오픈소스
